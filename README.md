# 🎉 Eid Family Games Website

A fun and interactive website created as a personal project to bring families and friends together during Eid.  
This project offers a collection of simple, enjoyable games designed to create laughter, friendly challenges, and memorable moments in a festive atmosphere.

The website includes multiple games that can be played with the whole family, making it perfect for gatherings and celebrations.

## 📝 Note
This project is continuously updated every Eid with new games, improved questions, and fresh ideas.  
The current version is not the final one — more enhancements and content will be added in future updates to keep the experience fun and exciting.
